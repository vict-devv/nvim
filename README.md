# Neovim
This is my personal [ Neovim ](https://neovim.io/) configuration.

## Usage
Clone or download this repo inside your nvim config folder, the most common location is at: `~/.config/nvim`

## Plugins
Here a list of the current plugins being used:

 - [ Alpha ](https://github.com/goolord/alpha-nvim): A nice greeter for neovim.
 - [ Autopairs ](https://github.com/windwp/nvim-autopairs): Helpful plugin for creating multiple characters auto-pairs such as [], () and "".
 - [ Bufferline ](https://github.com/akinsho/bufferline.nvim): Buffers/Tabs maganement.
 - [ nvim -cmp ](https://github.com/hrsh7th/nvim-cmp): A completion engine plugin for neovim. This requires extra plugins to provide snippets, e.g: *LuaSnip*.
 - [ Monokai ](https://github.com/loctvl842/monokai-pro.nvim): Monokai colorscheme.
 - [ Comment ](https://github.com/numToStr/Comment.nvim): Useful plugin for (un)commenting code.
 - [ Dadbod ](https://github.com/tpope/vim-dadbod): Vim plugin for interacting with databases (there're extra plugins to manage UI and autocompletion).
 - [ nvim -dap ](https://github.com/mfussenegger/nvim-dap): Debug Adaptar Protocol client implementation for neovim (there're extra plugins to manage UI).
 - [ GitSigns ](https://github.com/lewis6991/gitsigns.nvim): Helpful plugin to work with Git, providing signs and key mappings.
 - [ nvim -lspconfig ](https://github.com/neovim/nvim-lspconfig): Provides a basic LSP client configurations for various LSP servers.
 - [ Lualine ](https://github.com/nvim-lualine/lualine.nvim): A nice status line for neovim.
 - [ Mason ](https://github.com/williamboman/mason.nvim): Portable package manager for neovim, it helps to install LSP, DAP servers, linters and formatters.
 - [ Neotree ](https://github.com/nvim-neo-tree/neo-tree.nvim): A nice file system browser for neovim.
 - [ Noice ](https://github.com/folke/noice.nvim): Replaces the neovim UI, making it better. Provides pop-up menus for commands and notifications.
 - [ None -ls ](https://github.com/nvimtools/none-ls.nvim): Fork of the null-ls maintained by the community. This plugin helps non-LSP sources to hook into neovim LSP client.
 - [ Surround ](https://github.com/kylechui/nvim-surround): Helpful plugin to add/delete/change surrounding pairs.
 - [ Telescope ](https://github.com/nvim-telescope/telescope.nvim): Highly extendable fuzzy finder over lists.
 - [ ToggleTerm ](https://github.com/akinsho/toggleterm.nvim): Helpful plugin to manage the Terminal inside neovim.
 - [ Treesitter ](https://github.com/nvim-treesitter/nvim-treesitter): Language parser that provides a nice syntax highlight.
